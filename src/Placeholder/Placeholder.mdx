# Placeholder

Placeholder shows a spinner after a specified delay while content is loaded asynchronously.

<PropsTable of={Placeholder} />

## Loading

If `loading` is true, Placeholder will wait the delay before showing a spinner.

<Playground>
  <Placeholder delay={500} loading={true} />
</Playground>

## Content

Placeholder expects either a `React.Node` or a function that returns a `React.Node` as a child.

<Playground>
  <Placeholder delay={500} loading={false}>
    <span>Hey, I'm loaded asynchronously.</span>
  </Placeholder>
</Playground>

## Errors
 If an error occurs during an async call, pass an error message to the Placeholder component. It will call  `renderError` if provided, otherwise will fallback to default error UI.

<Playground>
  <Placeholder error="Some weird error occured..." />
</Playground>
