# Dropdown

Easily display contextual overlays using custom trigger elements. Dropdown's positioning system uses [Popper.js](https://github.com/FezVrasta/popper.js). Refer to their documentation for placement and option overrides.

## Props
<PropsTable of={Dropdown} />

## Example
<Playground>
  <Dropdown
    width={250}
    trigger={<Button variant="success">Open Dropdown</Button>}
  >
    {({ close }) => (
      <>
        <Dropdown.Header title="Dropdown" />

        <Dropdown.Body>
          <Dropdown.SectionTitle>Section One</Dropdown.SectionTitle>
          <Dropdown.Item>Item One</Dropdown.Item>
          <Dropdown.Item>Item Two</Dropdown.Item>

          <Dropdown.SectionTitle>Section Two</Dropdown.SectionTitle>
          <Dropdown.Item onClick={close}>Item One</Dropdown.Item>
          <Dropdown.Item disabled>Item Two</Dropdown.Item>
        </Dropdown.Body>

        <Dropdown.Footer>Footer</Dropdown.Footer>
      </>
    )}
  </Dropdown>
</Playground>
