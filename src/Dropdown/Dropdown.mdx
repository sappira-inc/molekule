# Dropdown

Easily display contextual overlays using custom trigger elements. Dropdown's positioning system uses [Popper.js](https://github.com/FezVrasta/popper.js). Refer to their documentation for placement and option overrides.

## Props
<PropsTable of={Dropdown} />
