# Icon

The material design icon library is included to render icons. See full list of icons here https://materialdesignicons.com/.

## Usage
To use `Icon`, first include the material design icon css in your index.html:

```
<head>
  <link rel="stylesheet" href="https://cdn.materialdesignicons.com/2.8.94/css/materialdesignicons.min.css" />
</head>
```
