# Formbot

Quickly build a form using a Formbot and pre-configured form components. Formbot wraps your form and manages the state providing neccessary callback methods via props.

## Props

<PropsTable of={Formbot} />

## Examples

### Formbot
<Playground>
  <Formbot
    initialValues={{
      name: '',
    }}
    validationSchema={{
      email: yup.string().email().required(),
      password: yup.string().required().min(4),
    }}>
    <Form>
      <Input name="email" label="Email" />
      <Input name="password" label="Password" />

      <Button type="submit" size="sm">
        Sign In
      </Button>
    </Form>
  </Formbot>
</Playground>
