# Formbot

Quickly build a form using a Formbot and pre-configured form components. Formbot wraps your form and manages the state providing neccessary callback methods via props.

## Props

<PropsTable of={Formbot} />

## Examples

### Formbot

<Playground>
  <Formbot
    validationSchema={{
      email: yup.string().email().required(),
      password: yup.string().required().min(4),
    }}>
    <Form>
      <Input name="email" label="Email" />
      <Input name="password" label="Password" />

      <Field>
        <Button type="submit">
          Sign In
        </Button>
      </Field>
    </Form>

  </Formbot>
</Playground>

### Async Validation

<Playground>
  <Formbot
    validationSchema={{
      random: yup.string().test('valid', 'This test was delayed by 2 seconds.', async function(value) {
        return new Promise((resolve) => {
          setTimeout(() => {
            resolve(false);
          }, 2000);
        });
      }),
    }}>
    <Form>
      <Input name="random" label="random" />

      <Field>
        <Button type="submit">
          Sign In
        </Button>
      </Field>
    </Form>

  </Formbot>
</Playground>
