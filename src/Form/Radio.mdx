# Radio

Form Radio component. Both font and icon sizes are configurable.

## Props

<PropsTable of={RadioGroup} />

## Examples

### Radio Group horizontal
<Playground>
  {() => {
    const radioValues = [
      {
        id: 1,
        value: 'radio1',
        name: 'radio1',
        label: 'Radio 1',
      },
      {
        id: 2,
        value: 'radio2',
        name: 'radio2',
        label: 'Radio 2',
      },
    ];
    return (
      <RadioGroup horizontal name="radio" id="radio" choices={radioValues} />
    )
  }}
</Playground>
