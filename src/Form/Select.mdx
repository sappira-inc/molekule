# Select

Form select component. Different sizes are available.

## Props

<PropsTable of={Select} />

## Examples

### Default Select
<Playground>
  {() => {
    const selectValues = [{ id: 1, value: 'male', label: 'Male' }, { id: 1, value: 'female', label: 'Female' }];
    return <Select id="select" name="select" options={selectValues} label="Example" floating />
  }}
</Playground>
