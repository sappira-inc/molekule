# Theming

Molekule comes with a default theme. You can override any of these configurations by passing a `theme` object to `<ThemeProvider />`

```js
{
  radii: [0, 2, 4],
  typography: {
    fontSize: 14,
  },
  colors: {
    primary: '#5759D6',
    green: '#1dd1a1',
    red: '#ee5253',
    blue: '#48dbfb',
    orange: '#feca57',
    grayLight: '#e9eef4',
    grayMid: '#8395a7',
    grayDark: '#576574',
  },
  variants: {
    success: 'green',
    danger: 'red',
    info: 'blue',
    warning: 'orange',
  },
  heights: {
    xs: 20,
    sm: 28,
    md: 32,
    lg: 40,
    xl: 44,
  },
  fontSizes: {
    xs: 10,
    sm: 10,
    md: 12,
    lg: 14,
    xl: 16,
  },
  breakpoints: [386, 768, 1024, 1440],
  grid: {
    containerMaxWidth: 1000,
    gutter: 15,
    columns: 12,
    sizes: {
      xs: 386,
      sm: 768,
      md: 1024,
      lg: 1440,
    },
  },
};
```
